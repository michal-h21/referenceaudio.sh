# Referenceaudio

This script can synchronize audio track from a video with audio recorded on an extenal microphone.

# Usage

    referenceaudio.sh videofile referenceaudio outvideofile

# Dependencies

- [Praat](https://www.fon.hum.uva.nl/praat/)
